Welcome to Picturesque's documentation!
===================================

Picturesque is a programming language, originally just made for drawing things using Turtle (hence the name, Picturesque). Picturesque's current version is 1.0.0-a1. This is a work in progress.

You can find it `here <https://github.com/tiashDev/pic-lang/>`_.
